# 📌 Overview

MeetingMind AI is an AI-powered meeting assistant designed to automate meeting documentation and improve productivity.

Instead of manually listening to long meeting recordings and writing notes, users simply upload an audio recording and the platform automatically:

- Transcribes the meeting
- Identifies speakers
- Generates structured summaries
- Extracts action items
- Highlights key decisions
- Saves meeting history
- Allows users to interact with meeting content

# ⚙ Environment Variables

### Backend (.env)

```env
PORT=5000

MONGODB_URI=

ASSEMBLYAI_API_KEY=

GEMINI_API_KEY=

CLIENT_URL=

NODE_ENV=development
```
### Frontend (.env)

```env
VITE_API_URL=your backend url/api
e.g. http://localhost:5000/api
```
